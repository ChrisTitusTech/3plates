This documentation set captures the product requirements, implementation rules, and setup steps for the 3plates app.

## What this project is

- One product across web, Android, and iOS.
- OAuth login with Google and Apple.
- Central storage for user progress, preferences, and account state.
- Contract-first client and backend integration.
